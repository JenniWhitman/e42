Converting from Hexadecimal to decimal is as easy as multiplying by 16... a lot.

1. Take the hexadecimal number and write it out, one digit per column.
2. Convert each hex digit to decimal. These should be second nature since A-F are equivalent to 10-15, but you can learn more on the Hexadecimal page.
3. Underneath this, write out 16x per column starting from the right where x begins with 0 and gains 1 per column (3, 2, 1, 0).
4. Multiply the number from steps 2 and 3 in each column together.
5. Add the resulting numbers together and you have the decimal equivalent of the given hexadecimal number.
