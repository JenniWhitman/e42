Converting from binary to hexadecimal is very easy as long as you can separate 4 bits (1s and 0s) at a time.

1. Take the string of binary and split it into groups of 4. If there are any missing bits, add zeros to the left side to make up the last group of 4.
2. Each of these values is equivalent to one hexadecimal digit from 0-F. Convert this using the Binary page as a guide for these small calculations or the Binary to Decimal Page if you'd like a little math practice.
3. Put these numbers together and you have a hexadecimal number equivalent to the original binary number.
Binary Changes Hexadecimal Exaplanation
1001010101 0010 0101 0101 Split the binary into groups of four after adding leading zeros to make this an even split.
0010 0101 0101 2 5 5 255 Convert each of the 4 values to a basic hex value. Since these are all under 15, this will easily convert to a 0-F. You can find a reference for these on the Binary page.
0010 0101 0101 255 Voila! 1001010101 in binary is equal to 255 in hexadecimal.
