Converting from binary to decimal is pretty easy... as long as you can multiply by 2!

1. Count out how long your binary number is.
2. Write out a table with columns - as many as the number you came up with in step 1.
3. Starting from the right, head each one with 2 to the power of 0 through (n-1) where n is the number from step 1.
4. In the row below, write out these values. This will come more naturally later, but is good practice for beginners.
5. Now, write your binary number out in the row below, one digit per box.
6. If a box has a 1, carry the number down from step 4, if not, carry down a 0. Write these values in this row.
7. Add the numbers across from step 6. This is the decimal equivalent of your binary number!

Our binary number is 11010011. It has 8 digits, so we have an 8 column table.
27 26 25 24 23 22 21 20
128 64 32 16 8 4 2 1 20 is equal to 1 in this case. It's easy to do the rest as you just keep multiplying by 2 each time. 1 times 2 is 2, 2 times 2 is 4, 4 times 2 is 8... easy pattern!
1 1 0 1 0 0 1 1 We enter our binary number across the table, one digit per cell.
128 64 0 16 0 0 2 1 We replace all of the zeroes from the row above into the base 2 row and come up with these values. In the next step we'll add them.
128 + 64 + 16 + 2 + 1 = 211 Voila! 11010011 in binary is equivalent to 211 in decimal.
